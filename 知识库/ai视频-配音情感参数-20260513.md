# AI视频配音情感参数指南

> 基于李一帆配音3步法则
> 版本：v1.0 | 日期：2026-05-13

---

## 核心法则

> "配音不是读稿，是演故事"
> 配音参数 = 标点 + 发声方式 + 场景语境

---

## 一、标点符号

| 标点 | 效果 | 适用场景 |
|------|------|---------|
| 感叹号！ | 加强强调感，声音上扬 | 重要数据、突破性消息 |
| 逗号， | 制造真实停顿，呼吸感 | 专业解读、理性分析 |
| 省略号… | 迟疑、悬念、情感延续 | 情绪铺垫、未尽之言 |
| 问号？ | 疑问、好奇、引导思考 | 提问式开场、悬念设置 |
| 波浪号～ | 轻快、活泼、亲近 | 温暖结尾、亲切互动 |

---

## 二，发声方式参数

### edge-tts参数

| 参数 | 中文说明 | 取值 |
|------|---------|------|
| Rate（语速） | 文字阅读速度 | -10% ~ +10% |
| Pitch（音调） | 声音高低 | -10Hz ~ +10Hz |
| Volume | 音量 | 默认 |
| Style | 语音风格 | news, cheerful, sad, angry等 |

### edge-tts支持的声音

| 声音 | 名称 | 适合场景 |
|------|------|---------|
| zh-CN-XiaoxiaoNeural | 晓晓 | 年轻、亲和、温暖 |
| zh-CN-YunxiNeural | 云希 | 专业、播报、中年 |
| zh-CN-Yunyang | 云扬 | 新闻播报、权威 |
| zh-CN-Xiaoyi | 小艺 | 年轻女性、亲切 |
| zh-CN-liaoning | 辽宁 | 东北口音、亲切 |
| zh-CN-shaun | 青春 | 年轻感、活力 |

## 三、场景配音参数

### 封面开场

```python
voice = "zh-CN-YunxiNeural"
rate = "+0%"      # 标准播报语速
pitch = "0Hz"
style = "news"    # 新闻播报风格
```

### 数据/技术解读

```python
voice = "zh-CN-YunxiNeural"
rate = "-10%"     # 语速放慢，让数据清晰
pitch = "0Hz"
style = "narration"  # 叙述风格
# 强调数据时用"。"停顿，数字前后略微停顿
```

### 新药/政策突破

```python
voice = "zh-CN-YunxiNeural"
rate = "+5%"      # 略带紧迫感
pitch = "+5Hz"    # 音调微升，表达重要性
style = "newscast"  # 新闻播报+权威
```

### 温暖收尾

```python
voice = "zh-CN-XiaoxiaoNeural"
rate = "-5%"      # 语速放慢，温暖感
pitch = "0Hz"
style = "affectionate"  # 亲切、温暖
```

### 金句/感悟

```python
voice = "zh-CN-YunxiNeural"
rate = "-15%"     # 极慢，让情感沉淀
pitch = "-5Hz"    # 音调下沉，厚重感
style = "sad"      # 略带情感，谨慎使用
```

## 四、李一帆配音公式

### 第一步：标点控制语气

```text
原文："今天有一个重要消息"
改写："今天，有一个重要消息——"
（用破折号制造悬念感）

原文："这个数字增长了很多"
改写："这个数字，增长了——很多！"
（用停顿+感叹强调数据）
```

### 第二步：发声方式替代情绪词

```text
不要写："用激动的语气说"
要写："语速加快+10%，音调上扬+5Hz，尾音加重"

不要写："用温柔的声音"
要写："语速放慢-10%，音调轻柔，尾音延长"
```

### 第三步：添加场景语境

```text
同一句"你终于来了"：
- 场景：患者康复出院
  → 语气：温暖+感动，语速放慢
  
- 场景：医生宣布治疗方案
  → 语气：专业+关怀，语速中等

- 场景：新药获批公告
  → 语气：权威+振奋，语速稍快
```

## 五、早报配音脚本格式

```markdown
【封面1-远景开场】
[语速+0%, 音调0Hz, 声音=云希, 风格=新闻]
台词："各位朋友大家好，这里是刘一的健康早报"
停顿：逗号处自然停顿

【封面5-数据预告】
[语速+5%, 音调+5Hz, 声音=云希, 风格=newscast]
台词："今天有3条重要资讯，干货很多——"
强调："3条"用重音

【新闻1-内容展开】
[语速-10%, 音调0Hz, 声音=云希, 风格=叙述]
台词："第一个消息，卡罗林斯卡医学院…"
专业词汇前后停顿，让观众消化

【结尾-温暖收尾】
[语速-5%, 音调温暖, 声音=晓晓, 风格=affectionate]
台词："关注健康，精算未来"
尾音轻柔，延长"未来——"
```

## 六、edge-tts代码示例

```python
import asyncio
from edge_tts import Communicate

async def generate_voice(text, output_path, voice="zh-CN-YunxiNeural", rate="+0%", pitch="0Hz"):
    """生成配音文件"""
    communicate = Communicate(
        text=text,
        voice=voice,
        rate=rate,
        pitch=pitch,
    )
    await communicate.save(output_path)
    print(f"Generated: {output_path}")

# 示例调用
asyncio.run(generate_voice(
    text="各位朋友大家好，这里是刘一的健康早报。今天有3条重要资讯，干货很多——",
    output_path="cover_voice.mp3",
    voice="zh-CN-YunxiNeural",
    rate="+0%",
    pitch="0Hz"
))
```

## 七、实用技巧

1. **数据前后停顿**：让数字更清晰
2. **专业词放慢**：新药名、技术词减速10%
3. **结尾尾音延长**："未来——""希望——"用延长音
4. **新闻开场用"各位"**：建立正式感
5. **数字用重音**：增强记忆点
