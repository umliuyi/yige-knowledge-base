### 2. 品牌视觉手册提示词
```
为一家名为「UIED技术团队」的公司打造一套精致的多页品牌视觉手册(多张图片形式)。

品牌整体气质应当充满趣味、具有未来感、色彩鲜明，并且设计感强烈，
在"玩具般的活力"与"高端创意科技品牌的精致感"之间取得平衡。

视觉风格需采用大胆且具有实验性的品牌语言，包括:
高饱和亮色、简洁几何图形、现代字体设计、富有动感的视觉节奏，
以及略带超现实感的图像表达。

整体氛围应传递出:创造力、年轻感、节奏感、触感体验以及概念前卫性。

在表达上，需要融合消费级产品的趣味性、时尚品牌的呈现方式，
以及当代数字设计的语言体系。

品牌手册中应体现:
- 对产品细节的极致关注(product obsession)
- 多样化的视觉实验(visual experimentation)
- 包装设计探索(packaging exploration)
- 编辑风格摄影(editorial photography)
- 沉浸式零售体验(immersive retail)
- 跨平台品牌应用(cross-platform brand application)

最终呈现应构建一个自信、当代、统一且完整的品牌世界观
- 既易于理解与接近，又具备强烈的艺术指导感(art direction)
- 在视觉上保持高度一致性，同时展现出高端创意工作室级别的美学标准

严格控制输出之后的最小字体不能小于10px可识别无变形。
```

### 3. 电商详情页提示词（分3步）

**第一步：产品信息提取**
```
产品信息智能提取，请仔细分析我上传的产品图片,自动提取以下信息：

自动识别项目:
- 品牌名称识别（从包装/产品上识别品牌LOGO文字、识别中文品牌名和英文品牌名、提取品牌标志的设计风格）
- 产品类型判断（识别产品所属类别：服装/食品/电子/美妆/家居/宠物用品等）
- 识别具体产品名称（从包装文字或产品形态判断）
- 识别产品规格（尺寸、容量、重量等）
```

**第二步：生成详情页结构**
```
根据识别报告分10屏2:3竖版生成详情页提示词文案给到我。
```

**第三步：按屏生成图片**
```
根据以上识别报告，分10屏2:3竖版生成详情页。
```

## 二、UI/UX作品集提示词

```
我将上传一组 UI 设计图片。请基于图片内容，为我生成一套 Behance 风格 UI/UX Case Study 作品集页面方案。

整体要求：
- 页面统一为 16:9 横版
- 风格参考高质量 Behance UI/UX Case Study，专业、克制、有作品集感
- 不要做成宣传海报，不要花哨装饰
- 页面之间保持统一的字体、色彩、留白、栅格和排版语言

请先分析上传 UI 图片，推导：
- 产品类型判断
- 目标用户
- 核心页面与功能
- 可推导的用户流程
- 适合的作品集视觉风格

然后生成 10 页 Case Study 页面方案：

1. 封面页：项目名称、产品类型/年份/关键词、核心 UI 展示、一句话业务目标
2. 产品背景 + 设计目标：业务场景、用户痛点、设计目标、价值假设
3. 用户分析 + 调研：Persona、使用场景、核心需求、典型问题
4. 信息架构 + 交互流程：信息结构、主流程、关键交互路径
5. 核心界面展示：首页/关键界面、功能说明、局部交互细节
6. 功能拆解与模块展示：核心功能模块、模块说明、对应界面
7. 设计系统：颜色、字体、图标、卡片、按钮、基础组件规范
8. 整体界面整合：多屏展示、全局视觉节奏、栅格与一致性
9. 项目总结：设计成果、关键思考、后续优化方向
10. Thank You 页：感谢语、项目名称、少量 UI 视觉呼应
```

## 三、关键结构规律总结

### 提示词核心公式
```
角色定义 + 风格描述 + 具体要求 + 格式规范 + 限制条件
```

### 风格词汇库
- 专业类：专业/克制/高端创意工作室/Behance风格/艺术指导感
- 色彩类：高饱和亮色/几何图形/现代字体设计/鲜明/超现实感
- 氛围类：富有动感/年轻感/节奏感/前沿性/沉浸式
- 结构类：统一/一致/完整世界观/高度一致性

### 格式要求常用词
- 页面比例：16:9 横版 / 2:3 竖版
- 字体限制：最小字体不小于10px
- 输出数量：10页 / 分10屏
- 排版：栅格/留白/字体色彩统一

## 四、对我们制图的启发

### 精算师风格幻灯片提示词
```
为一位北美精算师打造一套每日健康早报幻灯片（7张图，9:16竖版）。

整体风格：
- 深藏青渐变背景（RGB: 20-80, 50-150, 80-160）
- 科技感与专业感并重
- 简洁克制，不要花哨装饰
- 统一的视觉节奏

页面结构：
1. 封面页：日期 + 核心主题（大字居中）
2-6. 新闻内容页：标题 + 要点 + 数据
7. 结尾页：账号信息 + 引导关注

字体规范：
- 标题：白色大字，居中或左对齐
- 正文：深灰色(3a3a3a)，20-24pt
- 数据标签：青蓝色背景(#3a6a7a)白色字

品牌元素：
- 左侧：深色渐变块（约1/6宽度）
- 右上角：蓝色logo条
- 底部：账号"刘一｜精算师聊健康"
```
