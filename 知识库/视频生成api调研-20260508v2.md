# 视频生成 API 全自动化调研报告

> **调研时间：** 2026-05-08
> **调研目的：** 找到一条从"说一句话"到"视频文件自动生成并保存到本地"的全自动路径
> **背景：** 可灵API返回404、即梦CLI在Windows有OAuth Bug、海螺MiniMax视频API需单独开通
> **调研结论先行：** MiniMax是目前唯一确认有完整视频生成API且文档齐全的可用方案。

---

## 一句话结论

**MiniMax（海螺AI国际版 platform.minimax.io）是目前最有可能实现"一句话 → 自动生成视频 → 保存本地"全自动化链路的平台。** 其API文档完整、接入方式清晰、支持回调通知下载视频，唯一问题是国内访问可能需要代理。

**完整API调用示例（文生视频）：**

```python
import os, time, requests

api_key = os.environ["MINIMAX_API_KEY"]
headers = {"Authorization": f"Bearer {api_key}"}

# Step 1: 创建视频生成任务
url = "https://api.minimax.io/v1/video_generation"
payload = {
    "prompt": "A tiktok dancer is dancing on a drone, doing flips and tricks.",
    "model": "MiniMax-Hailuo-2.3",
    "duration": 6,
    "resolution": "1080P",
}
response = requests.post(url, headers=headers, json=payload)
task_id = response.json()["task_id"]

# Step 2: 轮询任务状态（每10秒一次）
query_url = "https://api.minimax.io/v1/query/video_generation"
while True:
    time.sleep(10)
    resp = requests.get(query_url, headers=headers, params={"task_id": task_id})
    status = resp.json()["status"]
    if status == "Success":
        file_id = resp.json()["file_id"]
        break
    elif status == "Fail":
        raise Exception("Video generation failed")

# Step 3: 下载视频文件
dl_url = "https://api.minimax.io/v1/files/retrieve"
# 用 file_id 获取下载链接，然后 requests.get() 下载到本地
```

**支持的15种镜头控制命令：**
Truck left/right, Pan left/right, Push in/Pull out, Pedestal up/down, Tilt up/down, Zoom in/out, Shake, Tracking shot, Static shot

### 最终推荐：MiniMax（海螺AI国际版）

**唯一可行全自动链路：**

```
用户输入文案
    ↓
调用 MiniMax Text-to-Video API
(POST https://api.minimax.io/v1/video_generation)
    ↓
配置 callback_url 或轮询状态
(GET https://api.minimax.io/v1/query/video_generation)
    ↓
获取 file_id，下载视频
(GET https://api.minimax.io/v1/files/{file_id})
    ↓
本地保存 → 自动发布抖音
```

**为什么选MiniMax：**
1. ✅ **完整的三步API流程**：创建任务→查状态→下载视频
2. ✅ **支持Webhook回调**：无需轮询，任务完成自动推送通知
3. ✅ **注册对中国用户友好**：国内可直接注册，有免费试用额度
4. ✅ **文档完整**：有Python代码示例，API设计清晰
5. ✅ **价格可接受**：$0.19/768P/6s（Fast模型），折合人民币约1.4元/条
6. ✅ **支持图生视频**：如果需要先用AI生成图片再生成视频，可以串联
7. ⚠️ **潜在风险**：国内访问 api.minimax.io 可能需要代理

**具体实施路径（供一哥参考）：**
1. 注册 https://platform.minimax.io 获取API Key
2. 测试文生视频API，验证国内网络连通性
3. 若网络不通，使用代理/VPS中转
4. 配置callback_url接收完成通知
5. 编写自动化脚本：接收文案→调用API→等待回调→下载视频→发布抖音
