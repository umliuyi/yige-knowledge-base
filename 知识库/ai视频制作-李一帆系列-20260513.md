## 体系框架（5集精华）

### 第4集：3招"活人感"

| 招式 | 方法 | 效果 |
|------|------|------|
| 代入感 | 过肩镜头/第一视角 | 从"旁观者"变"参与者" |
| 空间感 | 前景+背景虚化 | 拉开层次 |
| 镜头运动 | 推拉跟 | 推近情绪、拉出信息 |

### 第5集：打破15秒魔咒（三步）

1. **固定角色** — 统一人物描述
2. **设定场景** — 统一环境描述
3. **剧情连贯** — 保持叙事逻辑

### 第6集：电影感 = 景别 + 光影

**景别（叙事节奏）：**
- 远景：交代环境
- 中景：跟随动作
- 近景：传递情绪
- 特写：放大细节

**光影（氛围塑造）：**
- 冷色顶光：紧张
- 逆光：浪漫
- 底光：恐怖
- 丁达尔光：神圣

### 第8集：16种运镜

| 运镜 | 用途 |
|------|------|
| 推镜头 | 强化画面信息，突出情绪 |
| 拉镜头 | 揭示画面信息，展开场景 |
| 跟镜头 | 跟随人物，增强代入感 |
| 360度环绕 | 强调主体 |
| 上摇镜头 | 揭示情绪 |
| 希区柯克变焦 | 紧张/悬疑 |

### 第9集：情绪表情（2步）

1. **拆分表情** — 大情绪拆解为微表情
   - 公式：面部情绪 = 嘴角状态 + 眼部状态 + 面部状态
   - 例：开心 = 嘴角轻轻上扬 + 眼神变得柔和 + 眼睛微微弯起

2. **加入肢体动作** — 让情绪有载体
   - 公式：情绪状态 = 表情细节 + 手部动作 + 身体姿态 + 视线方向

## 完整制作流程

```
叙事逻辑（剧情连贯）
    ↓
角色统一（固定角色描述）
    ↓
场景统一（设定场景描述）
    ↓
视听语言
    ├── 景别：远景 → 中景 → 近景 → 特写
    ├── 光影：顶光/逆光/底光/丁达尔光
    └── 运镜：推/拉/跟/环绕/摇
    ↓
情绪细节（表情拆分 + 肢体动作）
```

```
文字内容 → 选风格 → 生成图片（单张）
```

**应该改成：**
```
封面（远景 · 乐城天际线 · 逆光 · 交代情绪）
    ↓
新闻1（中景 · 数据画面 · 推镜头 · 强化信息）
    ↓
新闻2（特写 · 药瓶/设备 · 丁达尔光 · 放大细节）
    ↓
结尾（全景 · 收尾 · 拉镜头 · 揭示全貌）
```

## 关键原则

1. 不要把所有信息塞进一个镜头
2. 拆分信息，让画面引导注意力
3. 光影决定故事氛围，景别决定叙事节奏
4. 情绪要拆分 + 肢体配合
5. 镜头运动要有目的（推近情绪/拉出信息）
