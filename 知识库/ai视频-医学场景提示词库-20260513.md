## 三、运镜关键词

| 中文 | 英文 | 用途 |
|------|-------|------|
| 推镜头 | push in, zoom in | 强化信息、突出情绪 |
| 拉镜头 | pull out, zoom out | 揭示全貌、展开叙事 |
| 跟镜头 | follow, tracking shot | 代入感、跟随人物 |
| 环绕 | 360° orbit, circling | 强调主体、戏剧性 |
| 摇镜头 | pan up/down | 揭示情绪、展开场景 |
| 希区柯克变焦 | dolly zoom, vertigo effect | 紧张、悬疑 |
| 航拍 | aerial drone shot, drone view | 展现规模、壮阔 |
| 微距 | macro shot, close-up detail | 放大细节、专业感 |

## 五、景别关键词

| 景别 | 英文 | 用途 |
|------|-------|------|
| 远景 | wide shot, establishing shot | 交代环境、规模 |
| 全景 | full shot | 完整人物+场景 |

| 近景 | close-up shot | 传递情绪、细节 |
| 特写 | extreme close-up | 放大细节、数据 |

## 六、综合示例

### 封面开场（乐城+精算师）

```
远景，Aerial drone view of Boao Lecheng medical zone at golden hour, 
modern glass buildings, palm trees, cinematic drone shot, 
documentary style, hopeful atmosphere
```

### 数据解读（图表特写）

```
特写，Medical data visualization on screen, line graphs showing 
treatment outcomes, blue-green clinical color scheme, 
overhead lighting, professional atmosphere, push in slowly
```

### 金句收尾（逆光剪影）

```
远景，Silhouette of professional walking towards window light, 
backlit by golden sunset, minimalist composition, 
cinematic documentary style, hope and dedication
```

### 新闻内容（新药场景）

```
中景，Modern pharmaceutical laboratory, scientist in white coat 
examining stem cell sample, blue-green ambient lighting, 
professional scientific atmosphere, documentary style
```

## 七、制作流程

```
1. 确定内容类型（封面/新闻/结尾）
2. 选择场景类型（建筑/设备/人物/数据）
3. 匹配光影（冷色=专业，暖色=关怀）
4. 选择运镜（推=强化，拉=揭示）
5. 组合英文提示词
6. 生成视频 + 配音 + 合成
```
