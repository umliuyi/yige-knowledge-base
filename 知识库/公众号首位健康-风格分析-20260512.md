## 二、标题分析

### 标题公式
```
[效果/情绪] + [技术名称] + [适应症/范围] + [状态词]
```

### 三篇文章标题对比

| # | 标题 | 公式拆解 |
|---|------|---------|
| 1 | 重获膝望！脐带间充质干细胞，逆转膝骨关节炎退变！ | 效果+技术+适应症+状态 |
| 2 | 华西乐城医院"自体NK细胞辅助免疫药物治疗实体瘤技术"正式启动！ | 机构+技术+状态 |
| 3 | 博鳌乐城29项生物医学新技术获批！细胞项目最新收费清单来了！ | 政策+范围+状态+实用信息 |

### 标题优点
- **有数字**：29项、具体技术名称
- **有情绪**：重获膝望、正式启动、来了
- **有机构背书**：华西乐城医院、博鳌乐城
- **有实用信息**：收费清单来了

## 三、排版格式分析

### 通用结构
```
1. 标题（超大号，情绪词突出）
2. 导语区（机构名+日期+信息来源）
3. 科普卡片区（灰色背景卡片，解释技术原理）
4. 核心内容区（白底，详细正文）
5. 机构信息区（医院介绍表格）
6. 知情同意书（底部免责提示）
```

### 排版优点
- **模块化设计**：每个部分独立成块，易读
- **科普卡片**：灰色背景突出，与正文区分
- **信息表格**：机构名称、地址、电话一目了然
- **知情同意书**：底部合规提示，提升可信度
- **留白充足**：不拥挤，阅读体验好

### 排版缺点
- 移动端适配性未验证
- 部分文章图片较小
- 知情同意书占篇幅较大

## 四、行文风格分析

### 行文公式
```
1. 情绪开头（标题吸引）
2. 技术背景介绍（权威来源）
3. 技术原理科普（通俗易懂）
4. 适应症说明（适用人群）
5. 机构信息（联系方式）
6. 合规声明（法律保护）
```

### 语言特点
- **权威感**：引用医院名称、研究数据
- **通俗化**：技术原理用通俗语言解释
- **情感共鸣**：标题用情绪词引发关注
- **合规意识**：底部知情同意书

## 五对一哥抖音内容的借鉴

### 可以学习的地方

| 方面 | 公众号做法 | 一哥可以借鉴 |
|------|-----------|------------|
| 标题 | 效果词+技术词+数字 | 精算数字+效果+情绪 |
| 结构 | 模块化分块 | 抖音字幕分条展示 |
| 科普 | 通俗解释复杂技术 | 用"算账"解释复杂问题 |
| 背书 | 机构来源权威 | 精算师身份+数据来源 |
| 结尾 | 引导行动（联系方式） | 引导关注、评论 |

### 不需要照搬的地方
- 知情同意书（抖音不需要）
- 详细联系方式（抖音是引流到私域）
- 过多机构信息（抖音要聚焦个人IP）

## 六、排版公式总结（完整版）

```
1. 标题（超大+情绪词）
2. 导语（机构+日期）
3. 科普卡片（灰底+图+通俗解释）
4. 正文（详细技术/产品介绍）
5. 权益/收费表格（白底+线条）
6. 机构信息表（联系方式）
7. 知情同意书（底部合规）
```

### 抖音借鉴
- 灰色背景卡片 → 抖音字幕强调框
- 表格化展示 → 抖音口播分条
- 机构信息 → 抖音引导关注
