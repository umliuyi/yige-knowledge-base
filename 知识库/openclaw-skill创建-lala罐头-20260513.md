## 核心内容

### 什么是Skill
Skill = 让AI在特定场景下自动执行特定任务的自动化能力

### Skill的核心要素
根据评论区整理：

| 要素 | 说明 |
|------|------|
| 触发条件 | 自动触发（可设定什么时候触发） |
| 执行动作 | 自动执行特定操作 |
| 工作流 | 串起来的一系列自动化步骤 |
