**发布机构：** Vertex Pharmaceuticals

---

### 核心数据（12例患者，12个月随访）

| 指标 | 结果 |
|------|------|
| **HbA1c达标（<7%）** | **12/12（100%）**达到ADA推荐目标 |
| **Time-in-Range（70-180 mg/dL）** | **>70%** 时间在目标区间 |
| **脱离胰岛素** | **10/12（83%）**在12个月时完全脱离外源胰岛素 |
| **胰岛素用量减少** | 平均减少 **92%** |
| **严重低血糖事件（SHE）** | **从第90天起全部消除** |
| **异基因移植物存活** | 葡萄糖反应性内源性C肽生成，12个月随访持续有效 |
| **主要终点达成** | 消除SHE + HbA1c改善 |

---

### 安全性

- 大多数不良事件为轻度或中度
- **无与zimislecel治疗相关的严重不良事件**
- 2例患者死亡，均与治疗无关
- 安全性与免疫抑制方案、输液程序、长期糖尿病并发症一致

**来源：** 搜索
