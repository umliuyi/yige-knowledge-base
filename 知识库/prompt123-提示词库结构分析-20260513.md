## 提示词结构公式

### 通用结构（4步）

```
Role（角色）
Background（背景）
Constraints（限制/要求）
Skills（技能）
```

### 示例分析

**英语单词记忆专家：**
```
Role: 英语老师，对单词记忆方法很有研究
Background: 教会记性差的用户快速记忆英语单词
Constraints: 解释要准确，联想要匹配单词含义，Excel格式输出
Skills: 词根词缀法/图像联想/场景记忆
Output: 表格格式
```

**小红书穿搭文案：**
```
Role: 小红书穿搭文案专家
Profile language: 中文
Description: 提供时尚灵感和文案，帮助用户分享穿搭风格
Skills: 捕捉时尚趋势/描述穿搭细节/提供搭配建议
```

## 对我的价值

### 可以直接用的提示词结构

**早报内容生成提示词：**
```
Role: 健康科普内容专家
Background: 面向普通受众，解释乐城医疗/干细胞最新进展
Constraints: 专业术语转白话，数字要具体，一句话说清价值
Skills: 精算师视角/类比解释/情绪共鸣
Output: 标题+正文+标签
```

**早报分镜提示词：**
```
Role: 视频分镜专家
Background: 医学科普视频制作
Constraints: 每镜头要有信息增量，避免堆砌
Skills: 景别选择/光影设计/运镜描述
Output: 分镜描述（英文提示词）
```

---

## 核心收获

**好的提示词 = 角色 + 背景 + 限制 + 技能 + 输出格式**

不一定要写得很长，结构清晰即可。
