## 一、爆款内容核心公式

```
爆款内容 = 【身份背书】+ 【热点/痛点选题】+ 【精算视角解读】+ 【外行听得懂】
```

**公式解读**：
- 身份背书：我有精算师背景，这个身份让我看得比普通人更深
- 热点/痛点：选用户真正关心的问题，而不是我想讲的
- 精算视角：用概率、风险、收益、长期账的框架分析
- 外行听得懂：不考试、不上课，把专业翻译成决策语言

## 二、开场公式（高留存率开场）

### 1. 反常识结论型 ✅验证
**公式**：颠覆用户既有认知 + 给出一个令人意外但有数据支撑的结论
**适用**：建立精算师"专业但反直觉"的人设
**例子**：
- "很多人都觉得自己不会得大病？精算师告诉你真实概率"
- "120万一针的CAR-T，但不是所有人都值得打"

### 2. 悬念前置型 ✅验证
**公式**：一个让人好奇的问题 + 暗示答案在后文
**适用**：热点追击、概率科普
**例子**：
- "保险利率要跌到历史最低！现在不买以后血亏？先别急，听我说完"
- "乐城为什么能用到国外新药？今天说清楚"

### 3. 数字冲击型 ✅验证
**公式**：具体数字 + 制造认知反差
**适用**：数据型内容、概率科普
**例子**：
- "超58万人爆仓！"（精算视觉）
- "每天1块钱，一年365块，得了大病能报销多少？"

### 4. 热点关联型 ✅验证
**公式**：政策/市场热点 + 我的精算视角解读
**适用**：时效性强的话题
**例子**：
- "央行降准降息落地，精算师告诉你保险该怎么配"
- "某款靶向药获批了，精算师来看值不值得等"

### 5. 场景代入型
**公式**：描述一个用户熟悉的场景 + 指出其中的逻辑漏洞
**适用**：产品对比、决策辅助
**例子**：
- "很多人买保险都先看孩子，但精算师告诉你这个顺序错了"

## 三、内容结构公式

### 三段式结构（通用）
```
[开场] 反常识/悬念/数字 → 抓住注意力
[展开] 精算逻辑拆解 → 给出分析框架
[收口] 结论 + 行动召唤 → 要么学到什么 / 要么关注我
```

### 算账式结构（精算师专属）
```
[问题] 这个决策值不值？
[精算分析] 概率 × 收益 vs 成本
[结论] 精算师的判断
```

### 揭秘式结构
```
[悬念] 这个东西背后的逻辑99%的人不知道
[拆解] 第一层… 第二层… 第三层…
[结论] 所以你应该这样看
```

## 四、结尾公式

### 引导关注型 ✅常用
"关注我，后面告诉你…"  
适用：系列内容第一集

### 引导私信型 ✅合规
"有具体情况可以私信我"  
适用：需要深度咨询的场景

### 金句收尾型
"最终回到一个核心问题——你愿意为健康投资多少？"  
适用：情绪共鸣型内容

### 反问收尾型
"所以你还觉得保险买不买无所谓吗？"  
适用：警醒型内容

## 五、精算视角体现公式

### 概率表达
- ❌ "大病很常见"
- ✅ "40岁男性，未来20年得大病的精算概率是XX%"

### 风险定价
- ❌ "这个保险贵"
- ✅ "按精算风险定价，这项保障的市场公允价值是XX，你现在付的价格是YY"

### 长期收益算账
- ❌ "存钱很重要"
- ✅ "现在每年存5000 vs 20年后大病支出50万，IRR算下来哪个更合算"

### 对比表达
- ❌ "出国就医好"
- ✅ "出国就医平均成本XX vs 乐城XX，时间成本、政策门槛、治愈率综合算下来，精算师给你的建议是…"
