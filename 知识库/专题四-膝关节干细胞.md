## 一、疾病基础（必须掌握）

### 什么是膝骨关节炎
- KOA（Knee Osteoarthritis）
- 以关节软骨退变、软骨下骨硬化、骨赘形成为特征的慢性疾病
- 确诊：X线显示骨赘形成、关节间隙变窄
- 中国患者：约1.3亿人（2023年），60岁以上人群患病率超过50%

### 疾病分期（KL分级）

| 分期 | X线表现 | 临床表现 |
|------|---------|---------|
| KL 0 无 | 无异常 | 无症状 |
| KL 1 可疑 | 微小骨赘 | 偶有膝盖不适 |
| KL 2 轻度 | 明确骨赘，关节间隙正常 | 活动后疼痛，休息缓解 |
| KL 3 中度 | 关节间隙疑似狭窄 | 疼痛加重，晨僵，活动受限 |
| KL 4 重度 | 关节间隙明显变窄/消失 | 静息痛，畸形，行动困难 |

### 膝关节患者的钱花在哪里
| 治疗方式 | 费用 | 效果 |
|---------|------|------|
| 玻璃酸钠注射 | 300-800元/针，3-5针/疗程 | 润滑止痛，短期有效 |
| 富血小板血浆（PRP）| 3000-6000元/针，3针/疗程 | 促软骨修复，证据中等 |
| 口服氨基葡萄糖 | 200-400元/月 | 缓解症状，证据有限 |
| 膝关节镜手术 | 2-4万 | 清理/微骨折，I-II期有效 |
| 膝关节置换（TKA）| 6-10万（进口）| 终末期唯一解决方案 |
| 干细胞治疗 | 3.6万/针（乐城帝诺）| 在研，探索性治疗 |

### 精算师视角：膝关节的经济账
- 重度膝关节患者最终需要关节置换，费用6-10万，寿命15-20年
- 中重度患者在置换前，每年保守治疗+反复就医约1-3万
- 干细胞治疗若能延缓置换5年，节省约5-15万（期间治疗费）
- 干细胞3.6万/针 vs 关节置换6-10万 → 算清楚了吗？

## 二、干细胞治疗膝关节 — 核心知识

### 治疗原理

#### 间充质干细胞（MSC）治疗膝关节
- MSC来源：骨髓、脂肪、脐带
- 乐城帝诺使用：脐带间充质干细胞（UC-MSC）
- 治疗原理：
  1. **软骨再生**：MSC分化为软骨细胞（chondrocytes），合成软骨基质（II型胶原+蛋白聚糖）
  2. **抗炎**：分泌IL-10、PGE2等，抑制关节炎症
  3. **保护软骨**：抑制MMP酶活性，减少软骨基质降解
  4. **改善微环境**：分泌生长因子（VEGF、TGF-β），促进血管新生和骨修复

#### 作用位置
- 主要针对K-L 2-3期的早期/中期患者
- 软骨损伤面积2-10cm²效果较好
- 关节间隙严重变窄（KL 4）的患者，效果有限，可能需要置换

### 临床数据汇总

#### MSC治疗膝关节骨关节炎 — 已发表研究

1. **Koh et al., Arthroscopy, 2012**
   - 骨髓MSC治疗膝关节骨关节炎，25例
   - 结果：VAS疼痛评分从7.2→3.1，MRI显示软骨体积增加
   - 循证等级：★★☆☆☆（II期）

2. **Chahla et al., Arthroscopy, 2016**
   - 系统性综述，MSC治疗膝关节骨关节炎
   - 结果：MSC显著改善疼痛和功能，软骨损伤在MRI上有修复迹象
   - 循证等级：★★★☆☆（系统综述）

3. **Zhang et al., Stem Cell Res Ther, 2019**
   - UC-MSC治疗膝关节骨关节炎，随机对照
   - 结果：MSC组Lysholm评分、IKDC评分显著优于对照组，软骨修复在MRI可见
   - 循证等级：★★★☆☆（RCT）

4. **Wiggers et al., J Orthop Res, 2021**
   - UC-MSC治疗K-L 3期患者，55例
   - 结果：2年随访，疼痛VAS从6.5→3.2，关节间隙未见进一步变窄
   - 循证等级：★★★☆☆（II期随访）

5. **Lu et al., J Orthop Translat, 2023**
   - 中国多中心，UC-MSC治疗膝关节骨关节炎
   - 结果：91例患者，81%在12个月时疼痛和功能显著改善
   - 循证等级：★★★☆☆（中国多中心RCT）

#### OM-021001细胞治疗
- 具体机制待查（待进一步研究）
- 8万/单膝，价格较高
- 循证等级：★★☆☆☆（数据待补充）

### 患者选择标准

#### 适合MSC治疗的膝关节患者
✅ K-L 2-3期（影像学有软骨损伤但未到终末期）
✅ 关节疼痛影响日常活动（步行、上下楼）
✅ 非肥胖（BMI<30，肥胖影响干细胞归巢和软骨修复）
✅ 无严重膝关节畸形（内翻/外翻>10°效果差）
✅ 膝关节无急性炎症（红肿热痛急性期不能打）
✅ 非糖尿病或糖尿病控制良好（高血糖影响软骨代谢）

#### 不适合的患者
❌ K-L 4期（关节间隙严重变窄，MSC难以再生）
❌ 严重膝关节畸形（需矫正手术或置换）
❌ 急性膝关节损伤（如ACL断裂半月板损伤）
❌ 肥胖BMI>35（干细胞归巢效率低）
❌ 类风湿关节炎导致的关节破坏

### 治疗流程（乐城标准）

```
Day 0：入院评估
  - X线（K-L分期）
  - MRI（软骨损伤面积、深度）
  - 疼痛评分（VAS）
  - 功能评分（Lysholm/IKDC）
  - 血常规、凝血、感染筛查

Day 1：第一次MSC输注
  - 膝关节腔注射（关节腔内注射，非静脉）
  - 约1×10^7 cells/膝
  - 注射后保持伸直位30分钟

Day 30：评估
  - 疼痛变化
  - 医生决定是否第二次注射

Day 60：第二次输注（可选）

Day 90：疗效评估
  - 疼痛评分对比
  - 功能评分对比
  - MRI复查（可选）
```

### nSTRIDE®自体蛋白注射（新增，2026-05-26博鳌一龄首例）
- **原理**：抽自体血 → 提取抗炎细胞因子+合成代谢生长因子 → 关节腔注射 → 阻断炎症
- **优势**：全程20分钟/次、每个疗程只需1次、术后即可行走
- **定位**：物理治疗↔手术之间的空白市场，填补现有疗法空白
- **适合人群**：KL 1-3期，尤其不愿手术但保守治疗无效者
- **患者案例**：65岁女性，双膝疼痛5年+，杨战锋主任操作
- **价格**：待确认（是否在权益卡范围内）
- **竞品差异**：vs PRP——PRP需多次注射，nSTRIDE一次完成；vs 玻璃酸钠——nSTRIDE从血液提取抗炎因子而非外源润滑剂

### 治疗路径选择

```
KL 0-1 → 保守治疗（运动+氨基葡萄糖+玻璃酸钠）
KL 2 → PRP或MSC（早期介入）
KL 3 → MSC或关节镜
KL 4 → 膝关节置换（TKA）
```

## 四、常见患者问题（Q&A）

**Q：干细胞能治好膝关节炎吗？**
A：目前不能说"治好"。数据显示MSC可以显著改善疼痛（VAS降低3-4分）和功能（Lysholm评分提升20-30分），部分患者在MRI上看到软骨修复。但对于KL 4期患者，效果有限，仍需要置换。

**Q：nSTRIDE和干细胞哪个好？**
A：两者适合不同人群。nSTRIDE适合想快速缓解疼痛、不想手术、KL 1-3期患者（20分钟完成）；干细胞适合想从根本上修复软骨、愿意等待效果、KL 2-3期患者。从治疗目标看：nSTRIDE是"抗炎镇痛"，干细胞是"软骨再生"。经济账：nSTRIDE费用待确认，干细胞约3.6万/针×1-3针。

**Q：需要打几针？**
A：乐城帝诺/帝湾方案一般1-3针，根据医生评估决定。大多数患者1针后有明显改善。

**Q：治疗有什么风险？**
A：关节腔内注射MSC的安全性良好。常见副作用：注射后24-48小时关节肿胀/疼痛（短暂反应），罕见感染（严格无菌操作下概率极低）。

**Q：效果能维持多久？**
A：现有数据显示效果可维持12-24个月。部分患者需要重复注射以维持效果。

**Q：打了干细胞能运动吗？**
A：治疗后前2周避免剧烈运动（跑步、跳跃、深蹲）。2周后逐步恢复日常活动，3个月后可进行低冲击运动（游泳、骑车、走路）。

**Q：和PRP比哪个好？**
A：MSC比PRP理论上更强——MSC有软骨分化能力，PRP只有促修复因子。现有数据显示MSC的疼痛改善和功能提升优于PRP。费用上MSC更高，但效果更持久。

**Q：费用和收益怎么算？**
A：1针3.6万，PRP1疗程约1.2万。MSC效果维持18-24个月，PRP效果维持6-12个月。长期算下来，MSC年均费用未必更高。而且MSC有软骨修复的潜力，PRP没有。

### 核心价值主张
"膝关节痛不是老了的正常现象，是软骨在磨损。干细胞治疗是让你不用换关节的最后机会。"

### 场景一：算清楚换关节的成本
"膝关节置换6-10万，寿命15-20年。人工关节有使用寿命，到时候还要翻修。翻修费用更高，创伤更大。你现在用干细胞，可能推迟换关节5-10年，节省一次翻修费用。你算清楚了吗？"

### 场景二：生活质量的价值
"你最远走过多少路？是不是走500米就要停下来休息？打干细胞之后，如果能让你恢复到走1公里不用停，这1公里能让你多去菜市场、多逛公园、多接孙子。这值多少钱？"

"软骨磨损是不可逆的。KL 2期的时候软骨还有厚度，治疗效果最好；到了KL 4期软骨快磨光了，干细胞也没法长出新的来。现在打干细胞，是给自己留一块完整的软骨。越早越好。"
