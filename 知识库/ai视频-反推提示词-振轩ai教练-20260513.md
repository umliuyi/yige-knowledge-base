## 核心方法

### 什么是"反推视频提示词"

把一段参考视频丢进AI工具 → AI分析出：
- 光影风格
- 构图方式
- 运镜/镜头语言
- 提示词

### 操作流程

```
参考视频 → 反推工具 → 提示词 → 用提示词生成新视频
```

### 应用场景

| 场景 | 用法 |
|------|------|
| 学习大佬的运镜 | 反推 → 模仿生成 |
| 快速出同款风格 | 反推 → 直接用提示词 |
| 研究爆款结构 | 反推 → 学习提示词逻辑 |
