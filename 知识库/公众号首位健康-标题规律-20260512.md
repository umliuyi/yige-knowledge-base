## 标题公式

### 核心公式
```
[机构/技术名称] + [数字/新消息] + [效果/情绪词]
```

### 情绪词库
| 类型 | 词 |
|------|---|
| 重磅词 | 重磅首发、正式发布、获批 |
| 效果词 | 一键唤醒、逆转、重获、修复 |
| 紧迫词 | 最新清单、来了、正式启动 |

### 技术词
- 干细胞、免疫药物、细胞治疗
- 乐城29项新技术
- 自体NK细胞、脐带间充质干细胞

## 对一哥抖音早报的借鉴

1. **标题公式可以套用：**
   - 机构+数字+技术+效果
   - 情绪词增加点击欲望

2. **选题来源：**
   - 每次乐城有新项目/政策，第一时间跟
   - 这个号就是乐城资讯的第一手来源

3. **内容方向：**
   - 精算师视角：算账/费用/值不值
   - 科普视角：新技朮/新政策解读
