# 龙虾团队日报规范

## 每日日报格式

每只龙虾每天在完成任务后，生成日报：

路径：`lobster-team/growth/daily/[龙虾名]/YYYY-MM-DD.md`

```markdown
# [龙虾名] 日报 | YYYY-MM-DD

## 今日做了什么
## 今日遇到的问题
## 今日学到什么
## 明日要主攻什么
```

## 执行原则
- **一开始就把标准定清楚**：第一次写日报前，先读这份规范
- **每天必须完成**：养成习惯，不写日报就是失职
- **主动发现问题**：写日报不是汇报流水账，是主动暴露问题、主动找解法
```

## 汇报关系
- 每只龙虾写完日报后，发送飞书通知一哥
- CEO小虾每晚汇总所有日报，有问题立即干预
